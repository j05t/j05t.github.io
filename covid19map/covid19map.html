COVID-19 Active Cases, Cumulative Deaths

Show description

This page shows COVID-19 active cases (white) and cumulative deaths (red) throughout the world. The height of the columns represents the number of active cases or deaths in that specific area. This project has been inspired by the World Population Density visualization by Jos Dirksen. To view this page you need to have a browser that supports WebGL (any recent version of Firefox, Chrome or Opera). This map is based on open data. The earth map is based on a download from the Nasa Earth observatory. COVID-19 data downloaded from github.com/CSSEGISandData/COVID-19 provided by CSSE at Johns Hopkins University. All dates are in the format year-month-day.
